# Tecnologias Utilizadas

- Bun: [Gerenciador de pacotes e runtime](https://bun.com/)
- ElysiaJs: [Framework para servidor. Alternativa ao Express/Fastfy](https://elysiajs.com/)
- Deploy na Vercel: [Plataforma para deploy serverless para backend e frontend](https://vercel.com/)
- Banco Postgres: [Deploy no Neon database](https://neon.com/)
- Prisma: [ORM para gerenciamento de banco de dados](https://www.prisma.io/)
- Zod: [Validação de schemas](https://zod.dev/)

# Elysia with Bun runtime

## Getting Started
To get started with this template, simply paste this command into your terminal:
```bash
bun create elysia ./elysia-example
```

## Development
To start the development server run:
```bash
bun run dev
```

Open http://localhost:3000/ with your browser to see the result.
